# AWS S3 Static Portfolio Website

A modern responsive portfolio website hosted using Amazon S3 Static Website Hosting.

## 🚀 Project Overview

This project demonstrates how to deploy a static portfolio website using AWS S3.

The website was built using:
- HTML
- CSS
- JavaScript

and deployed using:
- Amazon S3 Static Website Hosting

---

## ☁️ AWS Services Used

- Amazon S3
- Bucket Policy
- Static Website Hosting

---

## 🎯 Features

- Responsive Design
- Modern Dark UI
- Project Showcase
- Skills Section
- Lightweight & Fast
- AWS Free Tier Friendly

---

## 📚 Concepts Learned

- S3 Bucket Creation
- Static Website Hosting
- Public Bucket Policies
- Cloud Storage
- AWS Regions
- Object Hosting

## ⚙️ Deployment Steps

1. Create S3 Bucket
2. Upload Website Files
3. Enable Static Website Hosting
4. Configure Bucket Policy
5. Access Website Using Endpoint URL

---

## 🛡️ Free Tier Safety

This project is fully compatible with AWS Free Tier because:
- Uses only static files
- No backend compute resources
- Very low storage usage
