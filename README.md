# AWS S3 Static Portfolio Website

A modern responsive portfolio website hosted using Amazon S3 Static Website Hosting.

## 🚀 Project Overview

This project demonstrates how to deploy a static portfolio website using AWS S3.

The website was built using:
- HTML
- CSS
- JavaScript

and deployed using:
- Amazon S3 Static Website Hosting

---

## ☁️ AWS Services Used

- Amazon S3
- Bucket Policy
- Static Website Hosting

---

## 🎯 Features

- Responsive Design
- Modern Dark UI
- Project Showcase
- Skills Section
- Lightweight & Fast
- AWS Free Tier Friendly
